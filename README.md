# História do Dia

Reconstrução independente e educativa do arquivo **História do Dia**, com 366 histórias, materiais históricos recuperados e uma nova edição ilustrada integrada no texto.

Site: [atilasos.github.io/hisdodia](https://atilasos.github.io/hisdodia/)

## Transparência sobre inteligência artificial

As novas ilustrações da edição contemporânea foram geradas com inteligência artificial a partir do texto das histórias e de referências visuais recuperadas. O site identifica esta origem globalmente e dentro de cada edição ilustrada. As ilustrações históricas recuperadas são apresentadas separadamente como originais.

## Desenvolvimento

Requer Node.js 24 ou uma versão compatível.

```sh
npm test
npm run build
npm run serve
```

Para construir como GitHub project site:

```sh
SITE_BASE_PATH=/hisdodia npm run build
```
